# Caricature Images Guide

## How to Add Your Caricatures

Place your caricature images (with transparent backgrounds) in this folder with these exact filenames:

### Required Images:

1. **genie.png** - The Molayman genie from magic lamp (for homepage)
2. **welcome.png** - Friendly welcoming character (for login page)
3. **excited.png** - Excited/happy character (for signup page)
4. **teacher.png** - Teacher/professor character (for client dashboard)
5. **admin.png** - Authority/admin character (for admin dashboard)
6. **money.png** - Character with money/celebration (for application page)
7. **games.png** - Gaming/playful character (for games page)
8. **spin.png** - Lucky/excited character (for spin wheel page)
9. **chat.png** - Friendly talking character (for chat page)

### Image Specifications:
- Format: PNG with transparent background
- Recommended size: 400x400 pixels or larger
- Remove backgrounds using tools like remove.bg

### Currently Used Placeholders:
If an image is missing, a colorful emoji placeholder will appear instead.

### To Remove Background:
1. Go to https://www.remove.bg/
2. Upload your caricature image
3. Download the PNG with transparent background
4. Save to this folder with the correct filename
